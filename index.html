Research Summary

Patterns Found

Authentication

OAuth 2.0 is the dominant authentication method across most SaaS platforms.

API Surface

REST APIs are the most common, while GraphQL appears mainly in developer and productivity tools.

Self-Serve

Most modern SaaS platforms allow developers to create credentials without contacting sales.

Main Blocker

Enterprise products often require admin approval, paid plans, or enterprise contracts.

Applications

ID App Category Auth Self Serve API Buildable

Conclusion

Most SaaS products expose REST APIs with OAuth2 authentication, making them suitable for AI agent integrations. Enterprise products typically require additional onboarding, while modern developer platforms offer fast self-service access.
